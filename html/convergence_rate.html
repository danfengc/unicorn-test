Check the convergence rate of the MCMC chain¶

  • given a proposal distrubtion:

    • our goal is to check that for a given proposal distribution, whether the acceptance rate would change with respect to change in in prior distribution, or it would be a constant

  • given a prior distribution:

    • difference in acceptance rate with respect to change in proposal distribution
    • mcmc trace plot for different proposal distribution

δ checker

  • changes the value of δ while keeping other parameters fixed; the scale parameters for $\sigma^2$ and $\phi$ are denoted as $\theta_{\sigma^2}$ and $\theta_{\phi}$, respectively
    • $\beta$ prior ~ Generalized Logistic distribution(20, 30)
    • $\sigma^2$ prior ~ Gamma(4.0, 1.0)
    • $\phi$ prior ~ Gamma(5.0, 1.0)
    • $\kappa$ = 1
    • $\theta_{\sigma^2}$ = 1.0
    • $\theta_{\phi}$ = 1.0
